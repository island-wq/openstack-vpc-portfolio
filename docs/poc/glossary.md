# 용어집

| 용어 | 의미 |
|---|---|
| PoC | 도입 전 기술·운영 적합성 검증 |
| Provider Network | 물리 외부망과 직접 연결된 OpenStack Network |
| Tenant Network | Project 내부 Workload용 Overlay Network |
| DVR | Compute Node에 Routing 기능을 분산하는 Neutron 구조 |
| Floating IP | 외부 주소와 Instance 주소 간 1:1 연결 |
| Security Group | Instance Port에 적용되는 상태 기반 방화벽 정책 |
| Flavor | Instance의 vCPU·Memory·Disk 사양 |
| Cinder | OpenStack Block Storage 서비스 |
| Multipath | Storage 연결 경로 이중화 |
| Live Migration | Instance 실행 상태의 Compute 이동 |
| Host Evacuation | Compute 장애 후 다른 Host에서 Instance 복구 |
| Snapshot | 특정 시점 Volume 상태의 참조본 |
| Backup | 별도 저장소에 보관되는 Volume 복구본 |
| QCOW2 | QEMU/KVM 환경의 가상 Disk 형식 |
| VirtIO | KVM 가상머신용 반가상화 장치 Driver |
