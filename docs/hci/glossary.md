# 용어집

| 용어 | 설명 |
| --- | --- |
| HCI | Compute, Storage, Virtualization 기능을 동일 노드에 통합한 인프라 구조 |
| Control Plane | OpenStack API, Database, Message Queue, Scheduler 등 제어 서비스 영역 |
| Ceph | 여러 노드의 디스크를 묶어 Block·Object 스토리지를 제공하는 분산 스토리지 |
| ML2/OVN | Neutron 논리 네트워크를 OVN으로 구현하는 메커니즘 드라이버 구조 |
| Geneve | Tenant 네트워크를 물리 네트워크 위에 구성하는 Overlay 터널 프로토콜 |
| DVR | 라우팅 기능을 Compute 또는 Gateway Chassis에 분산하는 구조 |
| Gateway Chassis | OVN에서 외부 네트워크 연결과 일반 SNAT을 담당하는 노드 역할 |
| LACP | 여러 물리 링크를 하나의 논리 링크로 묶는 표준 프로토콜 |
| PXE | 네트워크를 통해 운영체제 설치 이미지를 부팅하는 방식 |
| IPMI | 서버 전원과 원격 콘솔을 관리하는 Out-of-Band 인터페이스 |
| N+1 | 구성 요소 1대가 실패해도 서비스를 유지하도록 예비 용량을 확보하는 방식 |
| Overcommit | 물리 자원보다 많은 가상 자원을 논리적으로 할당하는 정책 |
